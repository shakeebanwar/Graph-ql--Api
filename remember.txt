To fetch all the db data:

query {
  allBooks {
    id
    title
    author
    yearPublished
    review
  }
}


To fetch specific data:


query {
  book(bookid: 2) {
    id
    title
    author
  }
}


Creating a book:

mutation createMutation {
  createBook(bookData: {title: "Things Apart", author: "Chinua Achebe", yearPublished: "1985", review: 3}) {
    book {
      title,
      author,
      yearPublished,
      review
    }
  }
}
